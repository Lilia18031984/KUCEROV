# Random Quote Generator - Генератор случайных цитат

**Автор:** Кучеров Кирилл

## 📝 Описание

Random Quote Generator - это графическое приложение для генерации случайных цитат из предопределённого списка. Приложение позволяет добавлять новые цитаты, просматривать историю сгенерированных цитат, фильтровать их по автору и теме, а также сохранять историю в JSON-файл.

## ✨ Основные возможности

- 🎲 Генерация случайных цитат
- 📜 История всех сгенерированных цитат
- 🔍 Фильтрация истории по автору и теме
- ➕ Добавление новых цитат в библиотеку
- 💾 Автоматическое сохранение цитат и истории в JSON
- 🗑 Очистка истории
- 💿 Экспорт истории в JSON файл
- ✅ Валидация ввода при добавлении цитат
